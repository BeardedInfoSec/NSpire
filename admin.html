Paste a GitHub access token with Contents: read & write permission on BeardedInfoSec/NSpire. It stays on this device only. How do I get one?

1. Go to github.com/settings/tokens → Fine-grained tokens → Generate new.
2. Repository access → Only select → BeardedInfoSec/NSpire.
3. Permissions → Repository → Contents → Read and write.
4. Generate, copy the token, paste it below.
